# mythingslab.github.io

Technical documentation for the MyThingsLab fleet: the `my-things-core` SDK
plus every My[X] tool. Per-tool pages under `_tools/` are kept in sync with
each tool's README/CLAUDE.md by MyDocs — edit a tool's own repo, not the
pages here, to update its docs. See
[`my-things-core`](https://github.com/MyThingsLab/my-things-core) for the
shared SDK and build harness.
